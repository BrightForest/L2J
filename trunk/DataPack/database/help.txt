Данные для подключения к базе логин и гейм сервера находятся в data/options.ini

Принцип работы инсталлера:
В зависимости от выбранного пункта меню, открывается необходимая папка и из нее в базу устанавливаются все sql файлы.
При запуске пункта 2 Install game, автоматически запускается 3 Update game.
Для первой установки сервера необходима выбирать пункт 2, для дальнейших обновлений пункт 3.

Описание папок:
data/login - таблицы для логин сервера
data/game - таблицы для динамического контента(инфа о персонажах, кланах и т.д. и т.п.)
data/gameUpdate - таблицы для статического контента(спавн, дроп и т.д. и т.п.)
